## 1) 수출실적명세서

메뉴위치 : 회계 영세율수출관련서류 ▶ 수출실적명세서

### ▣ 개요

사업자가 Master L/C에 의해 재화나 용역을 직접 수출하는 거래에 대해서는 영세율을 적용합니다. 영세율 조기 환급 신고시 관세청에 수출신고를 한 후 재화를 외국으로 직접 반출하는 사업자인 경우에는 기존에 제출하던 각종 첨부서류 대신 수출실적명세서를 예정 및 확정 신고일 등 신고기간 단위별로 수출실적이 있는 경우 제출합니다.

### ▣ 입력방법

관세청에 수출신고를 후 외국으로 직접 분출(수출)하는 재화 이외의 영세율 적용분(국외제공용역 등)으로 세금계산서를 교부하지 않는 분의 건수, 외화금액, 원화금액의 합계를 입력합니다.
(참고 : 기타영세율은 하단 상세내역에서 건별 또는 총계를 입력합니다.)

**❶ 기간**
입력하고자 하는 기간의 월을 입력합니다. 1개월 또는 1~3월 등 기간단위로 조회 후 입력할 수 있습니다.

**❷ 수출신고번호**
수출신고서의 14자리 또는 15자리의 신고번호를 입력합니다.

**❸ 기타영세율 적용**
기타영세율 건수를 입력하며, 하단에 입력한 데이터는 화면상단의 ⑪기타영세율 및 인쇄물에 집계되어 반영됩니다.

**❹ 선(기)적일자**
수출재화(물품)를 실질적으로 선(기)적한 일자를 입력하며, 환입된 경우 환입일을 입력합니다.

**❺ 통화코드**
수출에 대한 대금을 결제 받기로 한 외국통화의 코드(영문자 3자로)를 직접입력 하거나 F2코드도움 또는 C아이콘으로 선택합니다.

**❻ 환율**
수출재화의 선(기)적일자에 한국외환은행에서 고시하는 매매기준율을 입력합니다.

**❼ 금액(외화)**
수출물품의 인도조건에 따라 지급 받기로 한 전체 수출금액으로 수출신고서의 (34)번 항목의 금액이며, 소수점 2자리까지 입력합니다.

**❽ 금액(원화)**
외화금액을 환율로 곱한 환산금액 또는 선(기)적일 전에 수출대금(수출선수금, 사전송금방식 수출 등)을 원화로 환가한 경우에는 그 금액을 원단위 미만은 절사하고 입력합니다.

### ▣ 편리한 기능(기능모음)

**❶ 마감**
수출실적명세서 전자신고 제출을 위해 조회된 기간에 저장된 데이터를 마감합니다.

**❷ 작성일자(가이드 영역)**
조회한 기간의 입력한 데이터를 작성한 년월일을 입력합니다. 조회기간의 마지막 월 기준으로 다음달 25일(신고기한)로 자동 반영되며, 사용자가 수정 가능합니다.

**❸ 데이터 정렬**
입력순, 선적일자순, 수출신고번호순으로 데이터 정렬합니다.

**❹ 엑셀서식 내려받기**
엑셀 업로드할 수 있는 기본양식을 내려받는 기능입니다.

**❺ 엑셀서식 불러오기**
내려받은 엑셀서식에 데이터를 입력하여 설정에 따라 업로드하는 기능입니다.

### ▣ 참고사항(원화로 수출한 경우 입력방법)

✜ 통화코드는 원화(KRW)를 선택하고, 환율은 1원으로 입력합니다.
✜ 외화, 원화란에 금액을 입력합니다.

## 2) 영세율첨부서류제출명세서

메뉴위치 : 회계 영세율수출관련서류 ▶ 영세율첨부서류제출명세서

### ▣ 개요

사업자가 Local L/C 또는 구매확인서에 의해 재화나 용역을 공급하는 경우 영세율을 적용하여 영세율 세금계산서를 발행해야 하는데, 이 경우 영세율첨부서류제출명세서도 같이 작성하여 부가가치세 신고시 제출해야 합니다.

### ▣ 입력방법

**❶ 구분**
1.명세서, 2.목록, 3.전체 항목 중 선택합니다.

**❷ 서류명**
개별소비세신고를 할 때 이미 제출한 서류의 명칭을 입력합니다.

**❸ Code, 발급자**
서류의 발급자를 F2코드도움 또는 C아이콘으로 입력합니다.

**❹ 발급일자**
서류의 발급일자를 입력합니다.

**❺ 선적일자**
수출재화(물품)을 실질적으로 선적한 일자를 입력합니다.

**❻ 통화코드**
수출대금을 결제 받기로 한 외국통화를 F2코드도움 또는 C아이콘으로 입력합니다.

**❼ 환율**
수출재화 선적일자와 외국환 거래시점을 위한 기준환율 또는 재정환율을 입력합니다.

**❽ 당기제출금액(외화)**
수출대금으로 받을 외화 금액을 소수점 4자리까지 입력합니다.

**❾ 당기제출금액(원화)**
입력된 환율과 외화금액을 기준으로 자동 계산되어 표기됩니다.

**❿ 당기신고해당분**
조회한 기간에 포함된 부가가치세 신고기간 내의 영세율 신고와 관련된 것으로, 외화와 원화를 직접 입력합니다.

전자신고시 제출사유란은 반드시 입력해야 하는 항목이며 조회영역의 구분을 1.명세서로 조회시 노출됩니다.

우측 가이드박스에서 거래기간, 작성일자, 제출사유를 입력합니다.

제출사유는 개별소비세 과세표준신고서와 함께 제출, 저장장치로 복사하여 제출 중에 선택합니다.

### ▣ 편리한 기능(기능모음)

**❶ 마감**
조회한 기간의 저장된 데이터를 부가가치세 전자신고를 위해 마감하며, 마감은 신고용 탭에서 구분(1.명세서)로 조회시 노출됩니다.

**❷ 데이터 정렬**
입력순, 구분순, 발급일자순, 선적일자순으로 데이터 정렬합니다.

### ▣ 참고사항

✜ 내국신용장 또는 구매확인서가 전자무역기반시설(U-Trade Hub)을 통하여 전자로 발급된 경우 내국신용장, 구매확인서 전자발급명세서만 제출합니다.

## 3) 내국신용장, 구매확인서 전자발급

메뉴위치 : 회계 영세율수출관련서류 ▶ 내국신용장, 구매확인서 전자발급

### ▣ 개요

전자무역기반시설을 통하여 개설되거나 발급된 내국신용장, 구매확인서에 대하여 작성 및 제출하는 부가가치세 부속서류로, 내국신용장과 구매승인서에 대해 서류번호, 발급일 등을 기재하여 신고할 수 있습니다. 매입매출 전표입력 메뉴에서 유형을 12.영세로 입력한 데이터 중 내국신용장과 구매확인서에 설정된 계정과목을 자동 반영하고, 서류번호와 발급일은 직접 입력합니다. 부가가치세 신고시 첨부서류로 제출하기 위하여 반드시 마감을 해야 합니다.

### ▣ 입력방법

기능모음 불러오기의 내국신용장, 구매확인서 항목에 입력된 계정과목에 대하여 매입매출전표입력 메뉴에서 12.영세로 입력한 데이터를 불러옵니다. 내국신용장, 구매확인서발급 메뉴는 부가가치세 신고기간 단위로 입력합니다.

**❶ 구분**
1.내국신용장과 2.구매확인서 중 발급한 서류를 선택합니다.

**❷ 거래일자**
거래일자를 입력합니다.

**❸ 서류번호**
내국신용장과 구매승인서에 표시된 서류번호를 직접 입력합니다.

**❹ 발급일**
내국신용장과 구매확인서의 발급일자를 직접 입력합니다.

**❺ 거래처명, 사업자등록번호**
F2코드도움 또는 C아이콘 클릭으로 거래처코드를 선택하면 거래처명과 사업자등록번호가 자동 반영됩니다.

**❻ 금액**
내국신용장 또는 구매승인서에 표시된 금액을 입력합니다.

### ▣ 편리한 기능(기능모음)

**❶ 불러오기**
매입매출전표입력 메뉴에서 유형을 12.영세로 입력한 데이터 중 내국신용장과 구매확인서에 설정된 계정과목을 불러옵니다.(계정과목코드 401~420, 901~930/세목사용시 40100~42099, 90100~93099)

**❷ 마감**
전자신고 제출을 위해 저장 데이터를 마감합니다.

**❸ 제출일자**
해당 서식을 부가가치세신고시 첨부서류로 제출할 년월일을 입력합니다. 조회기간의 마지막 월 기준으로 다음달 25일로 자동 반영되며, 사용자가 수정 가능합니다.

**❹ 작성일자**
해당 서식을 작성한 년월일을 입력합니다. 조회기간의 마지막 월 기준으로 다음달 25일로 자동 반영되며, 사용자가 수정 가능합니다.

**❺ 입력방법**
- **월별**: 1개월 단위로 입력하며, 조회는 부가가치세 신고기간에 상관없이 여러기간을 선택하여 조회할 수 있습니다. (부가가치세 신고시에는 신고기간을 조회하여 마감합니다)
  - ☞ (예시) 1월, 2월, 3월.. 12월까지 1개월 단위로 입력, 조회는 1월~12월까지 모두 조회 가능합니다.
- **기간별**: 부가가치세 신고기간 단위로 입력 및 조회됩니다.
  - ☞ (예시) 1월~3월, 4월~6월 부가가치세 신고기간 단위로 입력시 조회도 1월~3월, 4월~6월로 조회됩니다.
- **참고사항**: 월별/기간별로 입력단위를 선택 후 부가가치세 신고서 마감 또는 해당 서식 마감시 입력방식은 변경되지 않으며 해당 메뉴를 조회하기 전에 해당 기능이 활성화 됩니다.

**❻ 엑셀서식 내려받기**
엑셀 업로드할 수 있는 기본양식을 내려받는 기능입니다.

**❼ 엑셀서식 불러오기**
내려받은 엑셀서식에 데이터를 입력하여 설정에 따라 업로드하는 기능입니다.

## 4) 외화획득명세서

메뉴위치 : 회계 영세율수출관련서류 ▶ 외화획득명세서

### ▣ 개요

영세율 입증을 위해 작성하는 부가가치세 부속서류로, "부가가치세법 제 24조 및 동법시행령 제 31조부터 제 33조까지 규정하는 영세율 적용 재화 및 용역을 공급한 경우 제출"합니다.

### ▣ 입력방법

**❶ 기간**

## ★ BK 매핑·시사점

1. **영세율 매출은 매출전표 유형 12(영세)·16(수출)로 입력** → 본 16개 서식 대부분이 그 전표의 거래 세부유형(수출신고번호·L/C 서류번호·통화코드·환율·여권번호 등) 메타에서 파생. BK 매출전표 라인에 **영세율 증빙 메타 필드 그룹**(수출신고번호 14~15자리, 선적일, 통화코드, 환율, 외화금액, 내국신용장/구매확인서 번호·발급일)이 필요. 불러오기는 계정과목코드 구간(401~420·901~930, 세목 5자리 40100~42099·90100~93099)으로 매출전표를 집계.

2. **거래 유형별 동적 노출**(09·10과 동일 패턴): 직수출(수출실적), 간접수출(내국신용장·구매확인서·영세율첨부서류), 외화획득 용역, 외국인관광객 환급(면세점·미용성형·숙박·즉시환급), 선박/외항·항공 외국항행, 외교관 면세 — 해당 거래가 없는 테넌트엔 비노출. 업종·거래 프로파일 기반 카탈로그 gating.

3. **외화·환율 처리**가 영세율 서식의 공통 핵심: 통화코드(ISO 3자리)·선적일 매매기준율·외화금액(소수점 2~4자리)·원화환산(원단위 절사). 원화수출 특례(통화 KRW, 환율 1)도 지원. BK는 **환율 마스터(일자·통화·매매기준율)** 와 외화→원화 환산 유틸을 공통 서비스로 제공하고, 선수금·사전송금 환가 케이스를 별도 플래그로 처리.

4. **마감(전자신고 락) 공통**: 수출실적·영세율첨부·내국신용장 등은 전자신고 제출 위해 반드시 마감. 마감 시 입력 잠금·마감해제로 수정. BK 신고 워크플로의 상태(작성→마감→전자신고 제출)를 서식 공통 상태머신으로 모델링. 월별/기간별 입력단위 선택은 마감 후 변경 불가.

5. **외국인관광객 환급 서식군**(면세판매·즉시환급·미용성형·숙박)은 사후/사전 면세점 제도. 종사업장번호(단위과세 적용번호, 주사업장 0000)·면세판매장 지정번호·구매일련번호(20자리)·반출승인번호·환급창구운영사업자 등 면세점 특화 메타가 다수 → 사후면세점 테넌트용 별도 입력 프로파일. 미용성형·숙박은 가이드 미공개지만 카탈로그·전자신고 매핑은 유지.

6. **항공/선박 외국항행용역**(공급가액확정명세서·선박운송용역일람표)은 노선등록(10~99)·선박등록(1~999) 마스터 + 수입금액 구분(여객/화물/수화물/우편물/기타)이 키. BK는 운수업 테넌트용 **노선·선박 코드 마스터**와 수입 구분별 계정 매핑을 카탈로그 속성으로 제공.

7. **신고일자/작성일자 자동 기본값 공통 규칙**(09·10과 동일): "조회기간 마지막 월 + 다음달 25일" 자동 반영(수정 가능) → 신고 워크플로 공통 유틸. 부가세 신고기한 캘린더와 연동.

8. **엑셀 업로드 공통 패턴**: 다수 서식이 엑셀서식 내려받기/불러오기(거래처 자동등록·업로드 방식 설정) 지원 → BK는 서식별 엑셀 템플릿 스키마 + 검증/매핑 파이프라인을 공통 컴포넌트로 구현.

9. **영세율매출명세서는 총괄 명세**: 부가세신고서 과세표준 영세율 금액과 정합(reconciliation)되어야 함. 부가세법/조특법 조문별 영세율 공급실적을 집계 → BK 신고 엔진은 본 명세서를 신고서 영세율 라인의 근거 명세로 연결하고, 영세율 매출 존재 시 증빙 서식 미작성 경고(영세율과세표준불성실가산세 0.5% 리스크) 룰을 적용.
