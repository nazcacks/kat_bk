## 결산자료입력 상세 (★ 결산 모듈 설계 근거)

### 목적
결산 정리사항을 **수동 분개 없이 금액 입력만으로 결산분개를 자동 생성**.

### 화면
- 기간 설정(월별/분기/누적/기말), 결산정리사항입력, 결산 상세정보(매출액·당기순이익·소득평율)

### 자동분개 지원 항목
| 항목 | 입력/연동 |
|------|-----------|
| 재고자산 | 기말재고액 직접 입력 |
| 감가상각 | 「월별감가상각비계상」에서 자동 반영 |
| 대손상각 | 채권별 대손충당금 설정액 |
| 퇴직급여 | 귀속별 금액 |
| 무형자산상각 | 무형고정자산 상각액 |
| 준비금 | 환입액/전입액 |
| 법인세 | 기납입액 감안 추가 계산액 |

### 주요 기능
- 전표추가(결산분개를 일반전표입력에 자동 생성), 원가설정(F4, 매출원가 계정 연결), 감가상각비반영(F7), 계정과목코드보기
- **수동처리 필요**: 손익 예상/이연, 소모품 정리, 부가세 예수금·대급금 정리 등은 결산 월일의 일반전표입력에서 직접
- **재결산**: 수정 후 재입력 시 이전 결산분개 자동 삭제 후 재처리

### BK 시사점
- BK 결산 모듈은 ① 정리사항 금액 입력 → ② 자동 결산분개(`entryCode=CLOSING_DR/CLOSING_CR`) 생성 → ③ 재결산 시 멱등 재처리 구조를 채택.
- **제조원가명세서**는 결차/결대로 입력된 원가 대체 분개가 있어야 성립 → 01_전표관리의 결차/결대 경고 로직과 연계.
- **AI합계잔액시산표**는 BK의 AI 이상점수/경고칩과 자연 연계(이상 분개·잔액 탐지).
- 감가상각은 별도 고정자산 모듈(월별감가상각비계상)과 연동 → BK 고정자산(SA-FA)·결산 연계 설계.
