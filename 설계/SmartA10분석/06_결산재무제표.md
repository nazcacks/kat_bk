## 1) 결산자료입력

### 개요

결산작업의 마지막 단계에서 "결산 정리사항을 수동대체 분개하지 않고, 본 메뉴에 해당 금액을 입력하여 결산분개를 자동 생성합니다."

### 메뉴 구성

#### 1) 결산자료입력

**① 기간** — 결산을 수행할 기간을 입력합니다.

- **월별/분기별/반기별 결산**: (1월~1월, 2월~2월) 등 특정 기간을 지정하여 결산 수행
  - 예시: 2~2월 선택 시, 1월 31일 기말재고가 2월 1일 기초재고로 반영되고 2월 말일 재고가 기말재고로 처리됨
- **기간 누적 결산**: (1월~2월), (1월~3월) 등 연도 초부터 누적된 형태로 진행
  - 예시: 1~3월 선택 시, 전기 기말재고가 당기 1월 1일 기초재고로 반영되고 3월 말일 재고는 4월 1일 기초재고로 반영됨
- **기말 결산**: 기말 현재로 재무제표를 확정하는 결산 (회계기간 종료월에 따라 인식)
  - 예시: 회계기간이 1~12월인 경우, 1~12월 기간 설정 시 직전 기수의 기말재고가 당기 1월 1일 기초재고로 반영됨

**② 결산정리사항입력** — 기말 재고자산의 재고액, 감가상각, 대손상각 등 결산정리 항목에 금액을 입력합니다.

> 참고: "결산자료입력에서 결산분개가 어려운 항목은 사용자가 [일반전표입력]메뉴에 수동으로 입력해야 합니다." (예: 손익의 예상과 이연, 소모품비 정리, 부가세예수금과 대급금 정리 등)

**③ 결산 상세정보** — 매출액의 합계와 당기순이익을 확인할 수 있으며, 매출액 대비 당기순이익의 소득평율을 확인합니다.
※ 당기순이익 ÷ 매출액 = 소득평율

#### 2) 기능키 / 더보기(기능모음)

**① 전표추가** — 결산정리사항을 모두 입력한 후 [일반전표입력]메뉴에 결산 분개를 추가하는 기능. "결산분개를 추가하시겠습니까?" 팝업에서 [확인] 클릭 시 [일반전표입력]메뉴에 결산분개가 추가되고, 화면의 '결산분개'란에 해당 금액이 집계됩니다. 월(중간) 결산의 경우 [전표추가] 시 "익월 차감분개 포함 추가"를 선택할 수 있으며, 체크하여 전표추가 시 익월의 1일자로 결산차감분개가 추가됩니다.

**② 원가설정 (F4)** — 매출원가를 설정하고, 해당 원가경비코드를 연결하여 조회. ※ 최대 3개까지 설정 가능하며, 동일한 표준원가를 중복 설정할 수 없습니다.
- **매출원가코드**: 설정하고자 하는 계정과목명을 두글자 이상 입력하거나, [F2] 코드도움 또는 [C] 아이콘을 클릭해 매출원가 계정과목을 입력
- **원가경비**: 매출원가에 연결할 원가경비를 설정
- **표준원가 선택**: [계정과목 및 적요등록]메뉴에서 매출원가 계정과목에 설정된 계정구분에 따라 자동 반영

**③ 감가상각비반영 (F7)** — [고정자산등록]메뉴에 등록된 유/무형자산에 대하여 [월별감가상각비계상]메뉴에서 계상한 자산별 월별 감가상각비를 [결산자료입력] 메뉴로 반영합니다. 불러온 데이터 중 [결산반영금액]을 사용자가 수정할 수 있으며, 자산별 반영[1.여, 0.부]를 선택 후 [결산반영]클릭 시 경비별 감가상각비가 자동 반영됩니다.
※ [월별감가상각비계상]메뉴의 '월별감가상각비 명세'탭 조회가 선행되어야 합니다.
※ 비활성화 된 계정과목은 [월별감가상각비계상]메뉴에 존재하나, 당기의 전표입력 메뉴 또는 [전기분 재무상태표]메뉴에 입력되지 않은 계정과목입니다.

**④ 계정과목코드보기 (Shift+F4)** — 계정과목명 앞의 계정과목 코드를 표기합니다.

**⑤ 우측마우스 > 계산기** — 계산기 기능을 실행합니다.

**⑥ 우측마우스 > 엑셀** — 화면에 조회된 결산자료입력 데이터를 엑셀로 변환할 수 있습니다.

### ▶ TIP _ [결산자료입력] 자동결산 지원 항목

- **재고자산 기말재고액의 입력**: 상품, 원재료, 부재료, 재공품, 제품 등의 기말 재고액을 각각 클릭하여 직접 입력합니다.
- **유무형자산의 감가상각비 입력**: 판매비와 관리비, 제조경비, 도급경비, 분양경비의 '감가상각비' 란에 당기 감가상각비로 계상할 금액을 직접 입력하거나, 더보기(기능모음)의 [감가상각비반영]에서 [결산반영]으로 자동 입력할 수 있습니다. ※ 자동입력의 경우 [월별감가상각비계상]메뉴의 '월별감가상각비 명세'탭 조회가 선행되어야 합니다. ※ 조회 시 계상된 감가상각비는 계정과목 및 경비구분에 따라 '결산반영금액' 란에 자동반영되며, 사용자가 수정할 수 있습니다.
- **퇴직급여의 입력**: 퇴직급여의 귀속에 따라 판매비와 관리비, 제조경비, 도급경비, 분양경비의 해당 란에 금액을 직접 입력합니다.
- **대손상각의 입력**: 판매비와 관리비의 대손상각 란에 각 채권별로 해당 대손충당금 설정액을 직접 입력합니다.
- **무형고정자산의 상각액의 입력**: 판매비와 관리비의 무형고정자산상각 란에 각 무형고정자산 해당 계정과목별 당기상각액을 입력합니다.
- **준비금 환입액/준비금 전입액의 입력**: 영업외 수익/영업외 비용의 준비금 환입액 및 전입액란에 계정과목별로 환입액과 전입액을 입력합니다.
- **법인세(소득세 등)**: 법인세 등의 '2)법인세 계상'란에 기 납입한 법인세 비용(이자소득세, 법인세 중간 예납액 등)의 금액을 감안, '1)법인세등'의 금액을 공제한 후 결산 시 추가 계산할 법인세액을 입력합니다.

#### 자동결산 미지원 항목

※ 아래 항목은 자동결산이 지원되지 않는 항목으로, 결산 종료일자의 [일반전표입력] 메뉴에 수동으로 직접 입력합니다.
1. 손익의 결산정리(수익과 비용의 발생, 수익과 비용의 이연)
2. 소모품과 소모품비의 정리
3. 단기매매증권의 평가
4. 현금시재 검토, 가계정의 정리
5. 부가가치세예수금과 부가가치세대급금의 정리

### ▶ TIP _ 재결산 및 결산분개의 삭제

**결산대체 분개의 일괄삭제**
1. 결산분개가 이루어진 해당 월의 [일반전표입력]메뉴에서 [Shift + F6] 또는 더보기(기능모음)의 [자동분개 일괄검색]을 클릭하여 일자를 설정한 후(결산종료 월일) 하단의 [전체취소]를 클릭합니다.
2. 이 후 [결산분개]의 [∨]클릭하여, '결산분개'항목만 체크한 뒤 하단의 [삭제]버튼을 클릭합니다.
3. '삭제확인' 팝업창에서 삭제할 항목을 다시 한 번 검토한 뒤 [삭제]를 클릭하여 결산대체 분개를 삭제합니다.
※ 일괄 삭제기능으로 삭제한 전표는 복원 불가하므로 유의하여 진행합니다.

**재결산으로 인한 일괄삭제** — [일반전표입력]메뉴에서 직접 삭제하지 않아도 기존 결산대체 분개를 삭제 후 수정 내역을 다시 전표추가 할 수 있습니다.
1. [결산자료입력]메뉴에서 기존과 동일한 결산월을 선택합니다.
2. 조회 시 "저장된 데이터를 사용하시겠습니까?" 문구에서, 전에 작업한 내용을 기초로 입력하고자 할 경우에는 '확인'을, 처음부터 다시 작업하고자 할 경우에는 '취소'를 선택합니다. ※ 전표입력에서 수정한 사항이 있을 때에는 반드시 '취소'를 선택해야 합니다.
3. 결산내역을 입력 후 상단의 [전표추가]를 클릭 시 "기존 입력 된 결산 분개 삭제 후 진행합니다" 문구에서 [확인]을 클릭하면 [일반전표입력]메뉴에 결산종료 월일로 대체되어 있던 결산분개를 프로그램이 자동으로 삭제 후 현재 수정 내역을 최종으로 결산 전표 추가합니다.

## 2) 합계잔액시산표

> 메뉴위치 : [회계관리] 결산재무제표 ▶ 합계잔액시산표

### 개요
입력 된 자료에 의한 기간별 합계잔액시산표를 조회 및 출력하는 메뉴로 과목별, 세목별, 제출용 탭으로 구성되어 있습니다. 대차차액이 발생한 경우 메뉴 실행 시 [대차차액 전표 조회] 화면이 실행되며, 해당하는 전표를 조회 및 수정할 수 있습니다.

### 1) 합계잔액시산표
- **① 과목별** — 회사 관리용 합계잔액시산표로서, 전표입력에서 입력한 전표의 분개내용에 따라 과목코드별 집계금액을 조회합니다.
- **② 세목별** — 회사 관리용 합계잔액시산표로서, 전표입력에서 입력한 전표의 분개내용에 따라 세목코드별 집계금액을 조회합니다. [환경설정]메뉴에서 '2.세목사용(5자리)'으로 설정한 경우 표시됩니다.
- **③ 제출용** — 기업회계기준의 합계잔액시산표로 [현금 및 현금등가물, 매출채권, 매입채무] 등의 통합계정은 기본 설정되어 있습니다. 이 외의 통합계정은 더보기(기능모음)의 [통합계정등록]에서 추가, 삭제, 수정 가능합니다.
- **④ 기간** — 조회하고자 하는 월, 일을 입력하거나, 달력을 클릭하여 설정할 수 있습니다.

### 2) 더보기(기능모음)
- **① 퇴직연금설정 (F7)** — 퇴직금제도와 확정급여형퇴직금제도를 병행하는 경우 특정 제도에서 발생한 초과자산을 다른 제도의 부채와 상계할 수 있는 기능. 각 구분에 따른 계정과목을 설정할 수 있지만, '퇴직금운용자산과 퇴직급여 충당 부채 상계여부'는 [재무상태표]메뉴에서 선택할 수 있습니다. [재무상태표]메뉴에서 설정한 상계 여부 값은 [합계잔액시산표]로 자동 연동됩니다.
- **② 영문설정 (Shift+F3)** — 화면 및 인쇄물의 영문 표시 여부를 설정. 과목별 영문 설정은 [계정과목 및 적요등록]메뉴의 '영문명'에 입력된 정보를 반영합니다.
- **③ 출력명칭설정 (Ctrl+P)** — [계정과목 및 적요등록]메뉴의 '출력용명칭'에 입력한 정보로 조회할지 여부를 설정합니다.
- **④ 원가손익차감계정표시방법 (Ctrl+K)** — 과목별 탭에만 적용, 비용계정(500~800번대)의 차감계정 금액 표기 방법을 설정. ㉠ 합계금액으로 표시: 비용계정과 차감계정의 잔액을 각각 표기. ㉡ 순액금액으로 표시: 차감계정의 잔액은 별도 표기하지 않고, 비용계정의 합계 잔액에 차감되어 표기.
- **⑤ 매출액(원가)순액표시설정 (Ctrl+L)** — 과목별 탭에서만 적용. ㉠ 매출액 순액표시여부 [부]: 매출액과 차감계정(매출환입및에누리, 매출할인 등)의 잔액을 각각 표기 / [여]: 차감계정의 잔액은 별도 표기하지 않고 매출계정의 잔액 합계에 차감. ㉡ 원가 순액표시여부 [부]: 매출원가와 차감계정(관세환급금 등)의 잔액을 각각 표기 / [여]: 차감계정의 잔액은 별도 표기하지 않고 매출원가의 잔액 합계에 차감.
- **⑥ 조회기간설정 (Ctrl+M)** — 메뉴 실행시 [기간]입력란에 설정될 기본일(현재날짜 또는 회계기간 종료일)을 설정. ※ 현재 시스템 날짜보다 회계기간 종료일이 작은 경우 회계기간 종료일로 조회됩니다.
- **⑦ 임대주택자산등록 (F4)** — 임대주택 자산을 별도 대분류로 기재하여 사용하고자 할 경우 설정. 설정한 대분류명과 계정과목은 유동자산과 비유동자산 사이에 별도의 항목으로 기재됩니다.
- **⑧ 통합계정등록 (Ctrl+F10)** — 제출용 탭에서 조회, 기업회계기준의 합계잔액시산표로 적용하기 위한 통합계정을 등록. 기본 제공항목이 이미 등록되어 있으며, 사용자가 수정/추가 가능. ㉠ 통합과목명 / ㉡ 구분(자산/부채/매출/비용) / ㉢ 사용여부 / ㉣ Code·계정과목.
- **⑨ 대분류명편집** — 분류명을 수정하고자 하는 경우 사용. 계정과목(중분류명)란을 더블 클릭하여 수정.
- **⑩ 원장조회** — 선택한 계정의 원장을 조회하며, [전표수정(F8)]버튼을 통해 분개를 수정, 삭제할 수 있습니다.
- **⑪ 계정과목코드보기 (Shift+F4)** — 계정과목명 앞의 계정과목 코드를 표기.
- **⑫ 우측마우스 > 계산기** — 계산기 기능 실행.
- **⑬ 우측마우스 > 엑셀변환** — 조회된 화면을 엑셀로 변환.

## 3) 재무상태표

### 개요
입력된 자료에 의하여 특정 시점의 재무상태표를 조회 및 출력하는 메뉴로 과목별, 세목별, (세목)관리용, 제출용, 표준용 탭으로 구성. 비교식으로 확인되는 전기의 데이터는 [전기분 재무상태표] 메뉴에 입력된 데이터를 반영. 조회월의 누적된 재무상태표를 조회할 수 있으며, [결산자료입력]메뉴에서 자동 결산을 하거나, [일반전표입력]메뉴에서 수동결산 분개를 직접 입력한 경우 재고자산의 기말재고액을 포함한 결산 재무상태표를 확인할 수 있습니다.

### 1) 재무상태표
- **① 과목별** — 회사관리용 재무상태표로서, 전표입력에서 입력한 전표의 분개내용에 따라 과목코드별 집계 금액을 조회.
- **② 세목별** — 세목코드별 집계금액 조회. [환경설정]메뉴에서 '2.세목사용(5자리)'으로 설정한 경우 표시.
- **③ (세목)관리용** — 회사관리용으로 전표입력의 분개내용에 준한 세목별 금액을 별도로 집계하여 표시. '2.세목사용(5자리)' 설정 시 표시.
- **④ 제출용** — 기업회계기준의 재무상태표로서 기본적으로 [현금 및 현금등가물, 매출채권, 매입채무] 등의 통합계정은 설정. 이 외 통합계정은 [통합계정등록]에서 추가/삭제/수정.
- **⑤ 표준용** — 회사구분(개인/법인)에 따라 표준양식으로 작성된 재무상태표. 더보기(기능모음)의 [법인유형설정]에서 선택한 유형의 표준재무상태표를 조회.
- **⑥ 기간** — 조회하고자 하는 월을 입력하거나, [∨] 클릭하여 설정.

> ▶ TIP _ 재무상태표 조회 시 참고사항: *미처분이익잉여금(미처리결손금)은 [이익잉여금처분계산서]메뉴의 미처분이익잉여금(미처리 결손금)과 일치해야 합니다.*

### 2) 기능키 / 더보기(기능모음)
- **① 경영정보** — 재무상태표, 손익계산서에 의거하여 분석한 데이터를 조회. 안정성·수익성·활동성분석으로 구분되며, 당기·전기·전전기 데이터를 비교 조회. ※ 항목별 상세는 [경영정보(안정성, 수익성, 활동성)분석]메뉴 매뉴얼 참고.
- **② 퇴직연금설정** — 퇴직금제도와 확정급여형퇴직금제도를 병행하는 경우 초과자산을 다른 제도의 부채와 상계. '퇴직금운용자산'과 '퇴직급여 충당 부채' 상계여부를 선택하여 [재무상태표]에 표기. ※ 상계여부 '여' 설정 시 '퇴직연금운용자산' 금액이 [계정별원장]메뉴의 잔액과 상이할 수 있음. ※ '퇴직연금(급여)충당부채'보다 '퇴직연금운용자산' 금액이 큰 경우 상계 후 차액은 '투자자산'으로 집계.
- **③ 영문설정** — 화면 및 인쇄물의 영문 표시 여부 설정.
- **④ 법인유형설정** — '일반 법인용'과 '금융 보험 증권업' 중 선택. 금융재무제표 대상 업종코드는 [참고]에서 확인.
- **⑤ 출력명칭설정** — '출력용명칭' 정보로 조회 여부 설정.
- **⑥ 임대주택자산등록** — 임대주택 자산을 별도 대분류로 기재. 유동자산과 비유동자산 사이에 별도 항목으로 기재.
- **⑦ 통합계정등록** — 제출용 탭에서 조회. ㉠ 통합과목명 / ㉡ 구분 / ㉢ 사용 / ㉣ Code·계정과목.
- **⑧ 재무비율보기 (Shift+F9)** — 자산 대비 재무비율을 조회, '금액' 란 옆에 소수점 둘째자리까지 비율 표기.
- **⑨ 전년대비증감액보기** — 전기 대비 당기의 증가 감소액과 비율을 화면에 표기.
- **⑩ 원장조회** — 선택한 계정의 원장 조회, [전표수정(F8)]로 분개 수정·삭제.
- **⑪ 계정과목코드보기 (Shift+F4)** — 계정과목 코드 표기.
- **⑫ 우측마우스 > 엑셀변환** / **⑬ 우측마우스 > 계산기**.

### ▶ TIP _ 재무상태표와 이익잉여금계산서의 잉여금(결손금) 차액 발생

- **③ 전년도 이익처분(처리)내역 분개를 하지 않은 경우**: ㉠ [전기분이익잉여금처분계산서]에 '265.미지급배당금'이 입력된 경우 당기 [일반전표입력]에 처분 내역 수동 분개 — 처분전 분개: (차) 375.이월이익잉여금 / (대) 265.미지급배당금, 351.이익준비금; 처분후 분개: (차) 265.미지급배당금 / (대) 현금 및 현금성자산(101.현금 또는 103.보통예금). ㉡ 반대로 [일반전표입력]에 처분내역이 입력됐는데 [전기분이익잉여금처분계산서]에 미지급배당금이 없는 경우 → 미지급배당금 입력.
- **④ 중간배당 시 이익준비금을 분개처리한 경우** — 중간배당지급 분개: (차) 372.중간배당금 / (대) 현금 및 현금성자산; (차) 375.이월이익잉여금 / (대) 351.이익준비금. ㉠ 372.중간배당금은 [이익잉여금처분계산서]의 중간배당금란에 마이너스로 자동 반영되지만 351.이익준비금은 반영되지 않으므로 [이익잉여금처분계산서]에서 라인추가하여 수동 입력. 예시) '당기순이익'란에 커서를 두고 [라인추가] 클릭 후 생성된 라인에 계정과목코드 입력 → 금액란에 이익준비금 입력 → 계정과목코드 삭제 후 전표추가.
- **⑤ 경비코드를 잘못 입력하거나, 원가대체를 하지 않은 경우**: 제조원가(501~599)를 사용하는데 공사원가(601~699)코드를 사용한 경우 등 → [합계잔액시산표] 조회로 확인.
- **⑥ 수동 결산분개 시 원가대체 분개의 전표유형을 '결차/결대'로 입력하지 않은 경우**: 결산분개(매출원가 등)는 '5.결차'와 '6.결대'로 분개.

## 4) 손익계산서

> 메뉴위치 : [회계관리] 결산재무제표 ▶ 손익계산서

### 개요
일정 기간의 손익계산서를 조회 및 출력하는 메뉴로 과목별, 세목별, 세목(관리용), 제출용, 표준용, 포괄손익 탭으로 구성. 비교식 전기 데이터는 [전기분 손익계산서]를 반영. [결산자료입력] 자동결산 또는 [일반전표입력] 수동결산 분개 입력 시 매출원가를 포함한 결산 손익계산서를 확인할 수 있습니다.

### 1) 손익계산서
- **① 과목별** — 세목별 금액이 과목별로 집계되어 표시.
- **② 세목별** — 세목별 금액 표시. '2.세목사용(5자리)' 설정 시 표시.
- **③ 세목(관리용)** — 세목별 금액을 별도로 집계하여 표시. '2.세목사용(5자리)' 설정 시 표시.
- **④ 제출용** — 기업회계기준 손익계산서, 통합계정 기본 설정. [통합계정등록]에서 수정.
- **⑤ 표준용** — 회사구분(개인/법인)에 따라 표준양식으로 작성.
- **⑥ 포괄손익** — K-IFRS기준 당기손익정보와 기타포괄손익을 모두 포함. 더보기의 [법인세효과]에서 배열순서·집계계정 등록/수정 가능하며, [환경설정] 회계탭 [중소기업 회계기준 적용 및 재무제표 작성 선택]에서 '중소기업 회계기준 적용여부'를 [0.사용안함] 선택한 경우에만 조회.
- **⑦ 기간** — 조회 월 입력 또는 [∨] 선택.

### 2) 더보기(기능모음)

## 8) 현금흐름표

### 개요
기업의 일정기간 동안에 발생한 현금 흐름을 조회 및 출력하는 메뉴로, [손익계산서]가 제공하는 당기순이익 정보가 자금 흐름을 명확히 보여주지 못하는 단점을 보완하며, 현금흐름 및 변동 사항 등의 움직임을 명확히 파악.

### 1) 현금흐름표
- **① 기간** — 조회 월 직접 입력 또는 [∨] 선택.
- **② 현금흐름 분류** — [0.전체, 1.영업활동, 2.투자활동, 3.재무활동으로 인한 현금흐름] 분류 조회.
- **③ 당기** — 더보기의 [과목편집]에서 설정된 분류별 계정과목으로 당기의 전표입력 내용 반영. ※ 기타 현금 유출이 없는 비용, 기타 현금 유입이 없는 수익은 현금흐름표 계산 로직에 따른 계산 값을 반영.

> ▶ TIP - 기타 현금 유출이 없는 비용, 기타 현금 유입이 없는 수익의 계산
> ※ ④ > ⑦ : 기타 현금 유입이 없는 수익 / ※ ④ < ⑦ : 기타 현금 유출이 없는 비용
> 예시) ① 영업활동 현금흐름 = 732,124,827 − 7,500,000 = 724,624,827 (기타 현금 유출이 없는 비용은 차감, 수익은 가산) / ② 투자활동 = (−675,900,000) / ③ 재무활동 = 690,380,000 / ④ 현금의 증가(감소) = ①+②+③ {724,624,827 + (−675,900,000) + 690,380,000 = 739,104,827} / ⑥ 기말 현금 − ⑤ 기초 현금 = ⑦ 현금의 증가(감소) {886,774,827 − 140,170,000 = 746,604,827} → ④ 739,104,827 − ⑦ 746,604,827 = 7,500,000 → ④ < ⑦ = 기타 현금 유출이 없는 비용 7,500,000.

- **④ 전기** — 현금유출이 없는 비용 등의 가산, 현금유입이 없는 수익등의 차감은 [전기분 현금흐름표] 작성내역 반영. 그 외 항목은 [전기분 재무상태표] 내역을 불러와 반영.
- **⑤ 수정금액** — 재무제표에서 조회된 데이터가 적정한 현금흐름을 반영하지 못한 경우 최종적인 현금 유/출입액을 직접 설정. 예) 장부가액 100만원 건물을 150만원에 처분한 경우 B/S상 금액은 100만원이므로 50만원 합산.
- **⑥ 항목별합계액** — 현금흐름 분류별 합계와 최종 현금 증감 확인. ㉠ 영업활동: (당기순이익(손실) + 현금유출이 없는 비용 등의 가산 + 영업활동 자산·부채의 변동) − 현금유입이 없는 수익 등의 차감. ㉡ 투자활동: 유입액 − 유출액. ㉢ 재무활동: 유입액 − 유출액.

> ▶ TIP - 현금흐름표 작성 참고사항
> 1. 영업활동 현금흐름에서 매출채권 등의 충당금을 차감한 순액으로 분석.
> 2. 투자·재무활동 현금흐름은 유입·유출에 동일 계정과목이 동시 등록되어 있으므로 실제 발생한 현금 흐름 쪽을 남기고 반대쪽 계정과목 삭제.
> 3. 기초·기말현금흐름은 재무상태표(제출용)의 현금 및 현금등가물 데이터 반영.
> 4. 현금흐름표는 과목등록을 통해 과목수정·추가·재배열 가능.
> 5. 계정과목 증감은 제출용 재무상태표의 기초·기말 장부가액을 보여주며, 손익계산서 금액은 반영되지 않음.
> 6. 초기이월 현금흐름표는 비교식 작성용 입력이며, 당기분 현금흐름에 영향 없음.
> 7. 현금흐름 증가 요인은 자산의 감소, 부채의 증가. 예) 상품 판매(감소)와 선수금 증가는 현금흐름 증가.
> 8. 현금흐름 감소 요인은 자산의 증가, 부채의 감소. 예) 외상매출금 증가와 외상매입금 감소는 현금흐름 감소.
> 9. 미수금과 미지급금은 일반상거래 이외 거래의 채권·채무. 실무상 미수금 회수는 투자활동, 미지급금 지급은 재무활동으로 구분.
> 10. 단기차입금은 1년 이내 상환 부채로 신규차입·상환 거래가 빈번. 원칙상 총액 기재이나, 단기 만기 도래 시 순증감액 기재 가능.
> ※ 종료 시 "분석한Ⅰ+Ⅱ+Ⅲ의 합계와 대차대조표의 기말에서 기초현금및현금등가물을 차감한 금액과 일치하여야 합니다." 오류 발생 시 다시 작성.

> ▶ TIP - 현금흐름표 작성 예시
> **예시1) 매출채권과 대손충당금** — 매출채권 기초 110,000 / 기말 90,000 (대손발생 4,000), 대손충당금 기초 5,000 / 기말 6,000 (당기대손설정액 5,000), 매출채권(순액) 105,000 / 84,000.
> <직접법> 매출채권(총액)의 감소 20,000, 대손발생(채권감소) −4,000, 대손상각비(현금유출 없는 비용가산) 5,000, 차감계 21,000.
> <간접법> 매출채권(순액)의 감소 21,000.
> ※ 원천 분석과 순액 분석 결과는 동일하므로 실수 방지를 위해 순액으로 분석.
> **예시2) 재고자산과 재고자산평가손실** — 상품 기초 50,000 / 기말 60,000 (재고자산평가손실 4,000). <직접법> 상품의 증가(자산 증가→현금감소) −10,000, 재고자산평가손실 −4,000, 상품의 변동전 총액 −14,000. ※ 감모손실·평가손실이 있는 경우 재고자산 분석 필요.
> **예시3) 퇴직급여충당금과 퇴직금** — 퇴직급여충당금 기초 250,000 / 기말 230,000 (설정액 10,000, 퇴직 상계액 30,000). <직접법> 퇴직금 지급액 −30,000, 퇴직급여충당금 전입액 10,000, 순현금 흐름 −20,000. ※ 직접법 −20,000과 간접법(230,000−250,000=−20,000) 동일. 단, 기초퇴직급여충당금 초과 지급 시 직접 편집.

## 9) 결산부속명세서

> 메뉴위치 : [회계관리] 결산재무제표 ▶ 결산부속명세서 · [가이드 영상보기](https://www.wehago.com/westudio/watch/6Yi5eq)

### 개요
계정과목과 명세서 항목을 설정하여, 결산부속명세서를 생성하는 메뉴.

### 1) 조회월 및 화면구성
- **① 조회월** — 조회·작성 월 입력.
- **② 명세서명** — 최초 조회 시 [결산부속명세서 설정] > '명세서 추가 등록'에 설정된 명세서가 자동 작성. 더블 클릭/[F2]/[C] 시 등록한 명세서 선택·추가. 입력 명세서명은 결산부속명세서 출력 또는 세무조정계산서 전자책 변환 시 반영. ※ 미설정 명세서명은 붉은색(인쇄 불가), [결산부속명세서 설정]에서 수정된 경우 노란색(새로 불러오기).
- **③ 명세서항목** — [결산부속명세서 설정] > '명세서 항목'에서 설정한 데이터 형태에 따라 조회.

### 2) 툴바
- **① 소계/소계완료** — 일부 내역 소계 표기. [소계] 클릭 → 연속된 내역 체크 → [소계완료] 클릭 시 마지막 행 아래로 '소계' 생성. ※ [불러오기]/[데이터정렬] 변경 시 소계 삭제.
- **② 순서변경하기** — 명세서명 순서 또는 명세서 내역 순서 변경. [순서변경하기] 클릭 → 항목 클릭 → [▲][▼]로 변경 → [순서저장하기]. ※ 최초 불러오기는 '명세서 추가 등록' 순서. ※ [불러오기]/[데이터정렬] 변경 시 초기화.
- **③ 전체불러오기 (F3)** — 조회월 작성 명세서를 모두 삭제하고 [결산부속명세서 설정] 기준으로 새로 불러옴. ※ 특정 명세서 재반영 시 명세서 체크 후 액션바 [불러오기].
- **④ 재무제표일괄출력** — 재무제표 구분·유형·출력 재무제표 선택하여 일괄 PDF변환/인쇄. ㉠ 재무제표 구분(0.과목용, 1.제출용, 2.표준용, 3.세목용) / ㉡ 출력 유형(당기분/비교식) / ㉢ 출력 재무제표 선택. ※ 인쇄 시 미리보기 미지원, PC 기본 프린터로 출력.

### 3) 더보기(기능모음)
- **① 결산부속명세서 설정** — 명세서 추가 등록/삭제/수정. ㉠ 명세서 추가 등록([+] 클릭 시 최하단 등록, [⋮]로 편집/삭제) / ㉡ 계정과목(반영 계정 설정, 2개 이상 가능, [F2]/[C], 중복 설정 불가) / ㉢ 용지방향(1.가로, 2.세로) / ㉣ 명세서 항목(기본='1.기본항목(전표 데이터)'; 받을어음항목=[받을어음현황], 지급어음항목=[지급어음현황], 차입금항목=[차입금현황], 고정자산항목=[월별감가상각비계상_결산기준 감가상각 계상] 기준) / ㉤ 항목명(더블 클릭 수정) / ㉥ 사용여부 / ㉦ 전기명세서 불러오기(당기 설정 모두 삭제 후 전기 설정 반영, 이후 반드시 [전체불러오기]) / ㉧ 참고사항 / ㉨ 계정과목 삭제([F5] 또는 [계정과목 삭제]) / ㉩ 추가분 반영(추가 사용된 계정과목 명세서 자동 등록) / ㉪ 초기화 / ㉫ 확인(수정 시 [불러오기] 적용 필요).

## 10) 현금예금잔액검토

### 개요
현금과 예금 계정의 잔액검토를 위한 메뉴로 현금일괄잔액검토, 현금잔액검토, 당좌예금잔액검토, 보통예금잔액검토, 당좌예금잔액(세목), 보통예금잔액(세목) 탭으로 구성.

### 1) 현금일괄잔액검토 (기간별 현금계정을 자동으로 조정)
- **① 기간** — 1개월 최소 단위, 1년 치 한 번 검토 시 1월~12월 입력.
- **② 계정과목** — 잔액검토 사용 계정(개인=1.인출금, 법인=2.가지급금, 가수금).
- **③ 최소금액** — 검토 후 각 일자별 현금 보유액의 최소잔액 입력.
- **④ 최대금액** — 검토 후 각 일자별 현금 보유액의 최대잔액 입력.
- **⑤ 수정단위** — 발생시킬 가지급금/가수금(개인은 인출금)의 단위 입력.
- **⑥ 전표추가 (F7)** — 설정한 최소·최대금액·수정단위로 검토된 일별 수정내역을 [일반전표입력]에 추가.

### 2) 현금잔액검토 (월별 현금계정을 수동으로 조정)
- **① 검토월** — 검토 월 입력.
- **② 구분** — 1.출금, 2.입금 선택.
- **③ 계정과목** — 수동 조정 계정(1.가지급금, 2.가수금, 3.인출금) 선택, 전표추가 시 분개 사용. 법인 134.가지급금/257.가수금, 개인 338.인출금.
- **④ 수정금액** — 좌측 수정전·수정후금액 참고하여 입력. 입력 시 상단에 대표이사 가지급금·가수금 또는 대표자 인출금 잔액 확인.
- **⑤ 전표추가 (F7)** — '수정분개내역' 수동 입력 정보로 [일반전표입력] 전표 추가.

### 3) 당좌예금잔액검토 / 당좌예금잔액(세목) (당좌예금 계정을 통장과 비교, 누락 거래 입력·조정)
- **① 검토월** — 검토 월 입력.
- **② 계정과목** — [F2]/[C]로 검토할 당좌예금 세목 계정 설정. ※ 당좌예금잔액(세목) 탭에서만 표시.
- **③ 금융거래처** — [F2]/[C]로 검토할 당좌예금 거래처 입력.
- **④ 구분** — 1.출금, 2.입금, 3.차변, 4.대변.
- **⑤ 계정과목** — '1.출금'/'2.입금' 선택 시 [101.현금] 자동 설정. '3.차변'은 당좌예금 계정이 대변일 때 선택(차변 계정과목 코드 입력), '4.대변'은 당좌예금 계정이 차변일 때 선택(대변 계정과목 코드 입력). 이 경우 당좌예금 계정과목은 설정하지 않음.
- **⑥ 거래처** — '계정과목'란 설정 계정과목이 사용된 거래처 선택.
- **⑦ 적요** — '계정과목' 설정 계정과목의 적요내역 선택.
- **⑧ 수정금액** — 좌측 '당좌예금 일별현황' 참고 입력. 입력 시 예입금액/인출금액에 반영, 통장 잔액 확인.
- **⑨ 전표추가 (F7)** — '수정분개내역' 정보로 [일반전표입력] 전표 추가.

### 4) 보통예금잔액검토 / 보통예금잔액(세목) (보통예금 계정을 통장과 비교, 누락 거래 입력·조정)
- **① 검토월** / **② 계정과목**([F2]/[C], 보통예금잔액(세목) 탭에서만) / **③ 금융거래처** / **④ 구분**(1.출금, 2.입금, 3.차변, 4.대변) / **⑤ 계정과목**('1.출금'/'2.입금'→[101.현금] 자동, '3.차변'은 보통예금 대변일 때, '4.대변'은 보통예금 차변일 때; 보통예금 계정과목 미설정) / **⑥ 거래처** / **⑦ 적요** / **⑧ 수정금액**('보통예금 일별현황' 참고) / **⑨ 전표추가 (F7)**.

### 5) 더보기(기능모음)
- **① 대표자설정 (F3)(가지급금, 가수금)** — 전표전송 시 사용할 가지급금 대표자 설정. 설정값은 전표전송 시 [일반전표입력] 관리항목의 '사원'으로 반영.
- **② 우측마우스 > 계산기**.

## 11) AI합계잔액시산표

> 메뉴위치 : [회계관리] 결산재무제표 ▶ AI합계잔액시산표

### 개요
과거 회계 처리 이력과 빅데이터 분석으로 추천받아 생성된 각 증빙(전자세금계산서, 전자계산서, 신용카드, 현금영수증, 통장)의 수집 혹은 엑셀업로드 한 자동전표와 매입매출, 일반전표를 기준으로 AI합계잔액시산표가 조회.

### 1) AI합계잔액시산표
- **① 조회기간** — AI 분석을 통해 적용된 합계잔액시산표를 조회할 기간 입력.
- **② 미전송건수** — 자동전표처리 메뉴(전자세금계산서, 전자계산서, 신용카드, 현금영수증, 통장)에서 전표확정되지 않은 데이터 중 전표상태가 미추천, 확정가능, 중복전표, 삭제전표인 데이터를 각 계정별로 집계 표기.
- **③ 원장조회및수정** — 미전송 건수 표기 계정 더블 클릭 시 자동전표처리 메뉴에서 전표확정하지 않은 데이터 확인. 일자 더블 클릭 시 본 메뉴에서 계정별·전표 유형별 확인·수정·즉시 전표전송. 미전송 건수 미표기 계정 더블 클릭 시 [계정별원장] 실행되어 [일반전표입력]/[매입매출전표입력] 전표 수정·삭제·추가 입력. ㉠ 계정별원장 잔액: 현재 계정과목을 [계정별원장]에서 조회 시 확인되는 잔액 표기. ㉡ 전표전송: 자동전표처리 데이터를 [일반전표입력]/[매입매출전표입력]으로 전송.

## ★ BK 매핑·시사점

1. **결산자료입력 = BK 결산 자동분개 엔진.** 정리사항 금액 입력 → 결산분개 자동생성(`entryCode=CLOSING_DR/CLOSING_CR`, 전표유형 5.결차/6.결대) → 재결산 시 기존 결산분개 멱등 삭제·재생성. BK는 결산 세션(closingPeriodId)별로 자동분개를 묶어 재처리 시 일괄 롤백·재반영하는 트랜잭션 단위를 둔다.
2. **재고/감가/대손/퇴직/무형상각/준비금/법인세** 7개 자동결산 항목 vs **손익 발생·이연, 소모품, 단기매매증권평가, 현금시재·가계정, 부가세 예수금·대급금** 5개 수동 항목 → BK 결산 체크리스트에 자동/수동 플래그를 부여하고 수동 항목은 일반전표 직접입력 유도.
3. **감가상각비반영(F7)은 고정자산 모듈(월별감가상각비계상) 선행 조회 필요** → BK 고정자산(SA-FA)과 결산 모듈의 의존성·연동 순서 설계 근거. 결산부속명세서의 '고정자산항목'도 동일 메뉴 데이터 참조.
4. **재무제표 5탭 패턴(과목별/세목별/세목관리용/제출용/표준용)** 이 시산표·BS·PL·원가명세서에 공통 적용 → BK 보고서 엔진은 동일 데이터셋에 4~5개 집계 뷰(과목 3자리 vs 세목 5자리, 통합계정 매핑, 표준양식)를 매핑하는 단일 파이프라인으로 구현. 세목 탭 노출은 [환경설정] '2.세목사용(5자리)' 조건부.
5. **통합계정등록(제출용)** = 회사 계정과목을 기업회계기준 표준 항목에 매핑하는 사용자 정의 테이블 → BK는 테넌트별 `account_consolidation_map`(통합과목명·구분(자산/부채/매출/비용)·사용·Code) 제공.
6. **재무제표 간 정합성 규칙**을 검증 룰로 명문화: BS 미처분이익잉여금 = 이익잉여금처분계산서 미처분이익잉여금; PL 당기제품제조원가 = 제조원가명세서 당기제품제조원가; PL 당기순이익 = 이익잉여금처분계산서 당기순이익. 차액 발생 6대 원인(초기이월 오류/처분 전표추가 누락/전년 처분분개 누락/중간배당 이익준비금/경비코드 오입력·원가대체 누락/결차결대 미사용)을 BK 결산 진단 어드바이저로 자동 점검.
7. **AI합계잔액시산표 = BK AI 이상탐지·추천 연계의 핵심.** 미전송(미추천/확정가능/중복/삭제) 자동전표 건수를 계정별 집계하고, 미전송 포함/미포함 두 시나리오 시산표를 비교 제공 + 전기 대비 이상거래 탐지 → BK는 자동전표 파이프라인(02_자동전표처리)의 상태머신과 시산표 뷰를 직접 연결, 미전송 잔량을 결산 차단 게이트로 활용.
8. **현금예금잔액검토 = 통장 대사(reconciliation) + 가지급금/가수금/인출금 자동 조정.** 법인 134.가지급금/257.가수금, 개인 338.인출금. 최소·최대·수정단위 기반 자동 조정 전표 생성 → BK는 은행연동(오픈뱅킹/스크래핑) 잔액과 장부 잔액 차이를 조정 전표로 자동 제안하는 reconciliation 모듈로 매핑.
