## 4) 현장/프로젝트/부서 손익현황

> 메뉴위치 : 회계 보조장부Ⅱ ▶ 현장/프로젝트/부서 손익현황

### ▣ 개요
현장/프로젝트/부서탭으로 구성되어 있으며, 부문별손익계정에서 안분, 입력한 내용이 자동반영되어 각각의 부문별 손익계산서에 해당코드별로 집계되어 조회, 출력(비교식)됩니다. 반드시 부문별손익계정의 작업이 선행되어야 합니다.

### ▣ 조회방법
**도급현장별 탭** : 도급현장별로 안분, 입력된 손익계정과 금액이 월별로 표시됩니다.
- ❶ 기간 : 조회하고자 하는 기간을 입력합니다.
- ❷ 도급현장 : F2코드도움 또는 C아이콘으로 조회하고자 하는 도급현장을 선택합니다.

**분양현장별 탭** : 분양현장별로 안분, 입력된 손익계정과 금액이 월별로 표시됩니다.
- ❶ 기간 : 조회하고자 하는 기간을 입력합니다.
- ❷ 분양현장 : F2코드도움 또는 C아이콘으로 조회하고자 하는 분양현장을 선택합니다.

**프로젝트별 탭** : 프로젝트별로 안분, 입력된 손익계정과 금액이 월별로 표시됩니다.
- ❶ 기간 : 조회하고자 하는 기간을 입력합니다.
- ❷ 프로젝트 : F2코드도움 또는 C아이콘으로 조회하고자 하는 프로젝트를 선택합니다.

**부서별 탭** : 부서별로 안분, 입력된 손익계정과 금액이 월별로 표시됩니다.
- ❶ 기간 : 조회하고자 하는 기간을 입력합니다.
- ❷ 부서별 : F2코드도움 또는 C아이콘으로 조회하고자 하는 부서를 선택합니다.

### ▣ 편리한 기능(기능모음)
- ❶ 원장조회 : 선택한 데이터의 원장 내용을 조회하며, 전표수정을 통해 수정이 가능합니다.
- ❷ 계정과목코드보기 : 계정과목명 앞에 계정과목코드를 표시합니다.

## 6) 현장별 공사원가집계표

> 메뉴위치 : 회계 보조장부Ⅱ ▶ 현장별공사원가집계표

### ▣ 개요
조회기간의 현장별로 공사기간, 공사금액, 기성공사, 당기공사비내역을 조회 및 출력하는 메뉴입니다.

### ▣ 조회방법
- **❶ 기간** : 조회하고자 하는 기간을 선택합니다.
- **❷ 원가구분** : 0.도급공사원가, 1.분양공사원가 중에 선택합니다.
- **❸ 현장** : F2 코드도움으로 조회하고자 하는 현장을 입력하거나, 전체 현장을 검색하고자 하는 경우 공란으로 조회합니다.
- **❹ 기성내역**
  - 전기 기성액 : 전년도까지 총기성금액으로, 기능모음의 전기기성금액 조회를 클릭하면 수정 및 입력할 수 있습니다.
  - 당기 기성액 : 전표에서 입력한 공사수입금을 자동 반영합니다. 기능모음의 매출계정등록에서 추가 등록한 계정과목의 금액을 자동 반영합니다.
  - 총기성금액 : 전기 기성액과 당기 기성액을 합산한 금액을 자동 계산합니다.
- **❺ 공사비 내역**
  - 전기 공사비 : 미완성공사 초기이월 메뉴에서 입력한 내용을 자동 반영합니다.
  - 당기 공사비 : 전표입력 메뉴에서 입력한 재료비, 노무비, 경비 등을 합한 금액을 자동 반영합니다.
  - 총공사비 누적액 : 전기 공사비와 당기 공사비를 합한 금액을 자동 계산합니다.
- **❻ 당기 공사비 내역**
  - 재료비 : 전표입력 메뉴에서 입력한 도급원가의 재료비를 자동 반영합니다. 기능모음의 당기공사원가수정에서 재료비 내역을 수정할 수 있습니다.
  - 노무비 : 전표입력 메뉴에서 입력한 도급원가의 노무비 내역을 자동 반영합니다. 기능모음의 당기공사원가 수정에서 노무비 내역을 수정할 수 있습니다.
  - 경비 : 전표입력 메뉴에서 입력한 도급원가의 외주비 등 경비 내역을 자동 반영합니다. 기능모음의 당기공사원가수정에서 경비 내역을 수정할 수 있습니다.

### ▣ 편리한 기능(기능모음)
- **❶ 매출계정등록** : 기성내역의 당기기성액에 반영시킬 계정과목을 등록합니다.
- **❷ 원가설정** : 원가설정 매출원가를 설정하고, 해당 원가경비코드를 연결하여 조회할 수 있습니다.
  - 매출원가코드 : 매출원가코드 코드 란에서 입력하고자 하는 계정과목명 두 글자를 입력하거나, F2코드도움 또는 C아이콘 클릭으로 계정과목을 검색 후 입력합니다. 계정과목코드를 알고 있는 경우 직접 입력할 수 있습니다. (세목 사용시에는 코드가 5자리로 부여됩니다.)
  - 원가경비 : 원가경비 매출원가에 맞는 원가경비를 설정합니다.
  - 표준원가선택 : 계정과목 및 적요등록에 등록되어 있는 계정과목의 설정된 표준원가가 반영됩니다.
- **❸ 전기공사비 반영 설정** : 미완성공사초기이월에서 전기공사비를 불러오는 기준을 선택합니다.
- **❸ 당기공사원가수정** : 공사비내역의 조회된 데이터를 갱신하거나 원가를 수정합니다.
- **❹ 공사금액기준보기** : 공사기간 기준일과 공사금액기준 중에서 공사금액 불러오기 기준을 설정합니다.
- **❺ 총기성금액 이월** : 총기성금액을 차기로 이월합니다. 총기성금액은 차기의 전기기성액으로 반영됩니다.
- **❻ 현장별공사비 조회** : 현장별 전기와 당기 총공사 누적공사비를 조회합니다.
- **❻ 전기성금액 조회** : 기성내역의 전기기성액을 조회하며, 추가/수정/삭제가 가능합니다.
- **❼ 진행률 제외계정** : 공사비 계산 시 제외되는 계정과목을 설정합니다.

> (원문 기능모음 번호는 ❸·❻이 중복 표기되어 있으며 원문 그대로 옮김)

## 9) 현장/프로젝트/부서 거래처원장

> 메뉴위치 : 회계 보조장부Ⅱ ▶ 현장/프로젝트/부서거래처원장

### ▣ 개요
설정기간의 현장, 부서/사원, 프로젝트에 대해 조회한 계정과목의 거래처별 잔액 및 거래내용을 조회, 출력하는 곳으로 잔액, 내용, 총괄잔액, 총괄내용 탭으로 구성되어 있습니다. 거래처별 전기이월 금액은 전기분 거래처별 초기이월에 입력된 데이터가 반영됩니다.

### ▣ 조회방법
**잔액 탭** : 계정과목별로 선택된 거래처의 잔액명세서를 보고자 할 경우 선택합니다.
- ❶ 기간 : 조회하고자 하는 기간(해당월)을 입력합니다.
- ❷ 구분/구분코드 : 0.현장별, 1.부서별, 2.사원별, 3.프로젝트별 중 조회하고자 하는 구분을 선택하고, 해당 구분코드를 선택합니다.
- ❸ 계정과목 : F2 코드도움으로 조회하고자 하는 계정과목을 선택합니다.
- ❹ 거래처 : F2 코드도움으로 조회하고자 하는 거래처를 입력하거나, 전체 거래처 검색하고자 하는 경우 공란으로 조회합니다.

**내용 탭** : 각 구분별에 대한 해당 계정과목의 상세 거래내용을 각각의 거래처별로 보고자 할 때 선택합니다. 조회방법은 "잔액" 탭과 동일합니다.

**총괄잔액 탭** : 각 구분별에 대해 선택된 거래처별로 주어진 범위의 계정과목별로 집계된 리스트를 조회하고자 할 때 선택합니다.

**총괄내용 탭** : 각 구분별 선택된 거래처에 대해 거래처의 계정과목이 무엇이든 관계없이 해당 선택한 계정과목 범위 내에서 거래가 이루어진 순서대로 내용을 조회하고자 할 때 선택합니다.

### ▣ 편리한 기능(기능모음)
- **❶ 잔액표시방법** : 거래건별 잔액 표시 및 출력 여부 또는 잔액이 없는 경우 표시 여부를 선택합니다.
- **❷ 미등록 거래처 표기** : 전표입력시 거래처를 사용하지 않는 미등록거래처의 전표데이터 조회 여부를 선택합니다.
- **❸ 잔액집계** : 화면에 잔액이 있는 거래처만 조회할 지 여부를 설정합니다.
- **❹ 전표수정** : 내용 탭, 총괄내용 탭에 조회된 데이터 중 특정 자료에 대한 전표데이터를 조회하고자 할 때 사용합니다. 기능모음의 전표수정을 이용하거나, 조회하고자 하는 데이터를 더블클릭 하는 경우에도 동일하게 적용됩니다.

## 10) 프로젝트 누적손익현황

> 메뉴위치 : 회계 보조장부Ⅱ ▶ 프로젝트누적손익현황

### ▣ 개요
프로젝트별로 월별 손익항목의 내역을 조회합니다. 프로젝트별 누적 손익현황 초기이월의 내역은 전기 누적액에 반영되고, 당기 누적액은 전기 누적액과 당기 발생액을 합하여 반영합니다.

### ▣ 조회방법
- **❶ 구분** : 0.전체, 1.진행, 2.보류 중 조회하고자 하는 프로젝트 상태를 선택합니다.
- **❷ 프로젝트** : F2 코드도움으로 조회하고자 하는 프로젝트를 입력하거나, 전체 프로젝트를 검색하고자 하는 경우 공란으로 조회합니다.
- **❸ 조회기간** : 조회하고자 하는 기간을 입력합니다. 기간에 따른 데이터는 누적으로 반영됩니다.

### ▣ 편리한 기능(기능모음)
- **❶ 증감률 계산 설정** : 소수점 계산방법을 설정합니다.
- **❷ 원장조회** : 선택한 데이터의 원장 내용을 조회하며, 전표수정을 통해 수정이 가능합니다.
- **❸ 계정과목 코드보기** : 계정과목코드를 표시합니다.
- **❹ 전체프로젝트 인쇄** : 전체프로젝트를 한번에 인쇄하는 기능입니다.

## 11) 프로젝트 누적원가현황

> 메뉴위치 : 회계 보조장부Ⅱ ▶ 프로젝트누적원가현황

### ▣ 개요
프로젝트별 월별 원가항목의 내역을 조회하는 기능입니다. 초기이월 내역은 전기 마감 후 자동 반영되거나 직접 입력되며, 당기 누적액은 전기 누적액과 당기 발생액의 합으로 표시됩니다.

### ▣ 조회방법
- **❶ 기간** : 조회하고자 하는 기간을 입력합니다.
- **❷ 구분** : 0.전체, 1.진행, 2.보류 중 프로젝트 상태를 선택합니다.
- **❸ 프로젝트** : F2코드도움 또는 C아이콘으로 프로젝트를 입력하거나, 전체 프로젝트 검색 시 공란으로 조회합니다.
- **❹ 원가경비** : 50000번대 제조, 60000번대 공사, 70000번대 분양 중 해당 항목을 선택합니다.

### ▣ 편리한 기능(기능모음)
- **❶ 원가설정** : 매출원가를 설정하고 원가경비코드를 연결하여 조회할 수 있습니다.
  - 매출원가코드 : 계정과목명 두 글자 이상 입력하거나 F2/C아이콘으로 검색 후 입력합니다. (세목 사용 시 5자리 코드)
  - 원가경비 : 매출원가에 맞는 원가경비를 설정합니다.
  - 표준원가선택 : 등록된 계정과목의 설정된 표준원가가 반영됩니다.
- **❷ 증감률 계산 설정** : 소수점 계산방법을 설정합니다.
- **❸ 계정과목 코드보기** : 계정과목코드를 메뉴에서 표시합니다.
- **❹ 원장조회** : 선택한 데이터의 원장 내용을 조회하며, 전표수정을 통해 수정이 가능합니다.

## ★ BK 매핑·시사점

1. **부문(현장·프로젝트·부서) 축 = BK 6 관리차원** : 보조장부II는 전부 부문(현장·프로젝트·부서) 축의 원가·손익 관리. Smart A의 부문관리가 BK 6 관리차원(사업장·코스트센터·부서·프로젝트·현장·사원) 중 현장(SITE)·프로젝트·부서(DEPARTMENT)와 정확히 대응. DimensionConfig에서 부서(DEPARTMENT)를 독립 차원(C4 충돌해결)으로 둔 결정이 본 섹션 구조로 정당화됨.

2. **안분(배부) 규칙 엔진** : 원가안분·손익안분 메뉴(현재 SmartA는 가이드 미등록, 동영상 가이드만 제공)는 공통원가/공통손익을 부문으로 배부. BK는 배부기준(직접/비율/금액) 정의 + 결산 시 배부전표 생성으로 구현. 손익현황은 "부문별손익계정 작업(안분)이 선행되어야" 자동 집계 → 배부→집계 2단계 파이프라인.

3. **건설업 공사원가 모델(공사원가집계표·현장잔액)** : 도급/분양 원가구분, 기성내역(전기/당기/총기성), 공사비(재료비·노무비·경비), 미완성공사 초기이월, 총기성금액 차기이월, 진행률 제외계정 등 건설업 진행기준 회계의 핵심 데이터 구조. BK 건설업 모듈의 현장(SITE) 원가 모델로 채택. 매출원가코드+원가경비+표준원가 설정은 원가 분류 마스터.

4. **원가경비 코드대역** : 50000번대 제조 / 60000번대 공사 / 70000번대 분양. BK 원가 계정 체계의 대역 규칙으로 반영(01·06 결산과 정합).

5. **누적(프로젝트 라이프사이클)** : 프로젝트 누적손익·누적원가는 회계연도를 넘는 장기 프로젝트의 누적 집계(초기이월=전기 누적액, 당기누적=전기+당기발생). 프로젝트 상태(0.전체/1.진행/2.보류) 필터. BK는 프로젝트 차원에 회계기간 독립 누적 집계 + 상태 라이프사이클 제공.

6. **다차원 거래처 원장** : 거래처원장이 현장/부서/사원/프로젝트 구분 × 거래처 × 계정과목 교차 + 잔액/내용/총괄잔액/총괄내용 4탭. BK 다차원 거래처 원장으로 구현. 미등록 거래처 표기·잔액집계 옵션 포함.

7. **데이터 품질 관문** : 부문 원가/손익 리포트는 전표 라인의 차원 태깅이 정확해야 성립 → v2.1 우측 패널의 차원 필수입력 검증(`DIMENSION_REQUIRED`)이 데이터 품질의 전제. 거의 모든 메뉴가 원장조회·전표수정 드릴다운 제공.

8. **세목 5자리** : 원가설정의 매출원가코드가 세목 사용 시 5자리로 부여됨 → BK 세목(5자리) 모델과 정합.
