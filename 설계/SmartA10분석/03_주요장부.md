## 1) 전표출력

> 메뉴위치 : [회계관리] 주요장부 ▶ 전표출력

### ▣ 개요
[일반전표입력]메뉴와 [매입매출전표입력]메뉴에서 입력한 전표를 조회 및 출력하는 메뉴입니다. [일반전표입력]메뉴의 데이터는 1~50000으로 표시되고, [매입매출전표입력]메뉴의 데이터는 50001~99999로 표시됩니다.

### ▣ 조회방법

#### 1) 조회조건
**① 기간** : [일반전표입력], [매입매출전표입력]에 입력한 전표를 출력하기 위한 조회 기간을 선택합니다.

**② 구분** : 조회하고자 하는 전표구분(0.전체/1.출금/2.입금/3.대체/4.현금)을 선택합니다.

**③ 유형**
- 00.전체 : [일반전표입력]과 [매입매출전표입력]메뉴에서 입력한 모든 전표를 조회
- 01.일반전표 : [일반전표입력] 메뉴에서 입력한 전표를 조회
- 02.매입매출 : [매입매출전표입력] 메뉴에서 입력한 전표를 조회
- 03.현금조정 : [현금예금잔액검토] 메뉴에서 전표추가한 데이터를 조회
- 04.결산전표 : [결산자료입력] 메뉴에서 전표추가한 데이터를 조회
- 05.잉여금관련 : [이익잉여금처분계산서] 메뉴에서 전표추가한 데이터를 조회
- 06.손익계산서 : [손익계산서] 메뉴에서 전표추가한 데이터를 조회
- 07.중도시산표분개 : [중도시산표] 메뉴에서 전표추가한 데이터를 조회
- 08.원가안분분개 : [현장/프로젝트/부서원가안분] 메뉴에서 전표추가한 데이터를 조회
- 09.손익안분분개 : [현장/프로젝트/부서손익안분] 메뉴에서 전표추가한 데이터를 조회
- 11.물류/수금/지급분개 : [수금등록]과 [지급등록]메뉴에서 전표추가한 데이터를 조회
- 12.물류(매출/매입) : [매입마감회계처리]와 [매출회계처리]메뉴에서 전표추가한 데이터를 조회
- 13.복사전표 : [전표복사] 기능을 이용하여 복사한 전표를 조회
- 14.매출전표 : [매입매출전표입력] 메뉴에서 입력된 전표 중 유형이 매출인 전표를 조회
- 15.매입전표 : [매입매출전표입력] 메뉴에서 입력된 전표 중 유형이 매입인 전표를 조회
- 16.자동/슈퍼북(통장)전표 : [통장] 메뉴에서 전표전송한 전표를 조회

**④ 조회** : [∨]버튼 클릭시 하단으로 '조건검색'의 조건 화면이 노출되며, 조건을 설정하여 상세 검색 가능합니다. 조건 적용 시 분홍색 [∧] 버튼으로 표기됩니다.

▶TIP_조건검색의 [상대분개표시] 옵션 : 특정 조건에 부합하는 모든 분개(동일 전표번호)를 조회할 때 체크합니다. 예를 들어 [계정과목(811.복리후생비)] 조회 시, [부] 선택 시 811.복리후생비 라인만 검색되고, [여] 선택 시 상대 전표도 포함하여 조회됩니다.

#### 2) 기능키
**① 데이터 정렬** : 데이터 정렬 기준(입력, 차변-대변, 차변-차변(VAT)-대변-대변(VAT), 일자순, 계정과목순)을 선택합니다.

**② 전표출력옵션 (Ctrl+S)** : 인쇄옵션을 설정합니다.
- ㉠ 전표번호출력 : 설정값에 따라 전표출력시 전표번호 반영 여부가 결정됩니다.
- ㉡ 입.출금 전표 번호별 분리출력 : [0.여] 한페이지에 출력할 개수에 상관없이 출금전표 번호별로 1건씩 출력 / [1.부] 전표 1장 출력 계정수 설정 값에 따라 합산되어 출력
- ㉢ 전표 1장 출력 계정 수 : 입출금 전표 연속 출력 시 전표양식 한페이지당 반영할 개수(1~8)를 선택
- ㉣ 합계란 다음 장으로 이어짐 출력 표시 : 동일한 종류/일자의 전표인 경우 계정과목 개수를 초과 시 합계란에 "다음장으로 이어짐" 글씨가 표시되면서 다음 장에 전표가 출력
- ㉤ 계정과목 출력용 명칭사용 : [계정과목 및 적요등록]메뉴의 "출력용 명칭" 사용여부를 설정
- ㉥ 일반/매입매출전표 동일 인쇄설정 사용 : [일반전표입력]과 [매입매출전표입력]메뉴의 전표를 동일하게 인쇄 할지를 선택
- ㉦ 인쇄 시 부서/사원 반영 설정 : 인쇄 시 적요란에 반영할 부서 또는 사원을 선택

**③ 경비명칭변경** : 전표출력 시 원가 및 경비명을 변경하여 출력합니다.

## 2) 분개장

> 메뉴위치 : 회계 주요장부 ▶ 분개장

### ▣ 개요
[일반전표입력] 및 [매입매출전표입력] 메뉴의 하단부에 입력된 분개 내역을 조회 및 출력하는 메뉴입니다.

### ▣ 조회방법

#### 1) 조회조건
**① 기간** : [일반전표입력], [매입매출전표입력]에 입력한 분개를 조회하기 위한 기간을 선택합니다.

**② 전표**
- 0.전체 : [일반전표입력] 및 [매입매출전표입력] 메뉴의 모든 분개내용을 조회합니다.
- 1.일반 : [일반전표입력]메뉴에 입력된 분개내용을 조회합니다.
- 2.매출매입 : [매입매출전표입력]메뉴 하단부에 입력된 분개내용을 조회합니다.
- 3.매출 : [매입매출전표입력]메뉴의 유형이 '매출'인 데이터의 하단부에 입력된 분개내용을 조회합니다.
- 4.매입 : [매입매출전표입력]메뉴의 유형이 '매입'인 데이터의 하단부에 입력된 분개내용을 조회합니다.

**③ 선택**
- 0.전체 : 입/출금전표와 대체전표 구분없이 모두 조회합니다.
- 1.출금 : 출금전표의 분개내용을 조회합니다.
- 2.입금 : 입금전표의 분개내용을 조회합니다.
- 3.대체 : 대체전표의 분개내용을 조회합니다.

**④ 조회** : [∨]버튼 클릭시 하단으로 '조건검색'의 조건 화면이 노출되며, 조건을 설정하여 상세 검색이 가능합니다. 조건을 설정하여 조건검색한 경우 분홍색 [∧] 버튼으로 표기됩니다.

#### 2) 기능키
- **① 현금합산** : [매입매출전표입력]메뉴에서 분개 유형을 [1.현금]으로 입력한 데이터 중 동일한 전표번호에 현금을 분리하여 분개한 경우 현금계정 합산여부를 설정합니다.
- **② 경비명칭변경** : 인쇄 시 원가 및 경비명을 변경하여 출력합니다.
- **③ 우측마우스 > 엑셀변환** : 조회된 화면을 엑셀로 변환할 수 있습니다.
- **④ 우측마우스 > 계산기(Ctrl+Q)** : 계산기 기능을 실행합니다.
- **⑤ 전표수정 (F7)** : [일반전표입력], [매입매출전표입력]메뉴 실행없이 [분개장]메뉴에서 바로 분개를 수정할 수 있습니다. 전표를 더블클릭 하거나, 단축키(F7)을 클릭 시 하단 분개내역이 반영되며, 분개 수정 및 삭제가 가능합니다.
  - 주의사항 : 전표수정 기능은 더보기(기능모음)의 현금합산 > [현금합산 적용안함]으로 설정한 경우에만 사용 가능합니다.

## 3) 일월계표

> 메뉴위치 : [회계관리] 주요장부 ▶ 일월계표

### ▣ 개요
일계표 또는 월계표를 조회 및 출력하는 메뉴입니다.

### ▣ 조회방법

#### 1) 일계표
- **① 기간** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **② 구분** : [환경설정]메뉴 공통탭의 계정과목코드체계 > [1. 세목사용(5자리)]로 설정한 경우 [0.과목, 1.세목]을 선택합니다. [1.세목]으로 조회 시 사용자가 설정한 계정세목으로 조회가능하며, [0.과목]은 합산된 계정과목으로 조회됩니다.

#### 2) 월계표
- **① 기간** : 조회하고자 하는 기간의 월을 입력합니다.

#### 3) 기능키
- **① 임대주택 (F4)** : 임대주택 자산을 별도 대분류로 기재하여 사용하고자 할 경우 설정합니다. 설정한 대분류명과 계정과목은 유동자산과 비유동자산 사이에 별도의 항목으로 기재됩니다.
- **② 계정과목코드보기/감추기(Shift+F4)** : 계정과목명 앞의 계정과목코드를 표시하거나 감출 경우 사용합니다.
- **③ 원장조회 (F3)** : 선택한 계정의 원장을 조회하며, [전표수정(F8)]버튼을 통해 분개를 수정하거나 삭제할 수 있습니다.
- **④ 우측마우스 > 엑셀변환** : 조회된 화면을 엑셀로 변환할 수 있습니다.

## 4) 총계정원장

> 메뉴위치 : [회계관리] 주요장부 ▶ 총계정원장

### ▣ 개요
당기의 계정별 내역 및 잔액을 조회하는 메뉴로, 월별 또는 일자별로 현금을 포함한 모든 계정과목의 내역을 조회할 수 있습니다.

### ▣ 조회방법

#### 1) 월별
- **① 기간** : 현재 접속한 기수의 [회사등록]메뉴 '회계기간'이 자동으로 반영됩니다.
- **② 계정과목** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회하고자 하는 계정과목을 선택합니다.
  - 전체 계정과목을 조회하고자 하는 경우 별도 입력없이 [Enter]를 클릭합니다.
  - 복수의 계정과목을 조회하고자 하는 경우 물결 '~'을 사용하여 구간을 지정할 수 있습니다. ex) 101~103
- **③ 구분** : [환경설정]메뉴 공통탭의 계정과목코드체계 > [1. 세목사용(5자리)]로 설정한 경우 [0.과목, 1.세목]을 선택하여 조회합니다. [1.세목]으로 조회 시 사용자가 설정한 계정세목으로 조회가능하며, [0.과목]으로 조회시 합산된 계정과목으로 조회됩니다.

#### 2) 일별
- **① 기간** : 기간의 월/일을 사용자가 지정하여 조회합니다.
- **② 집계옵션 (F3)** : 데이터 조회시 집계항목인 [월계, 누계]를 화면에 표기할지 여부를 선택합니다.
- **③ 원장조회** : 선택한 계정의 원장 내용을 조회하며, [전표수정(F8)]버튼을 통해 분개를 수정하거나 삭제할 수 있습니다.

## 5) 현금출납장

> 메뉴위치 : [회계관리] 주요장부 ▶ 현금출납장

### ▣ 개요
현금의 수입과 지출에 대한 내용을 조회 및 출력하는 메뉴로, [전체, 부서별, 사원별, 현장별] 탭으로 구성되어 있습니다. 전표 입력 시 [2.입금, 3.차변(현금)]은 '입금'에, [1.출금, 4.대변(현금)]은 '출금'에 반영됩니다.

### ▣ 조회방법

#### 1) 전체
- **① 기간** : 조회하고자 하는 기간의 월/일을 입력합니다.

#### 2) 부서별/사원별/현장별
- **① 기간** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **② 부서/사원/현장** : 조회하고자 하는 관리항목을 설정합니다. 전체 관리항목(부서/사원/현장)으로 조회할 경우 코드없이 [Enter] 클릭합니다.

#### 3) 기능키
- **① 전표수정 (F7)** : [일반전표입력], [매입매출전표입력]메뉴 실행없이 [현금출납장]메뉴에서 바로 분개를 수정할 수 있습니다. 수정하고자 하는 전표를 더블클릭 하거나, 단축키(F7)을 클릭 시 하단 분개내역이 반영되며, 분개 수정 및 삭제가 가능합니다.
- **② 집계옵션 (F3)** : 데이터 조회시 집계항목인 [일계, 월계, 누계]를 화면에 표기할지 여부를 선택합니다.
- **③ 잔액표시방법 (Ctrl+Z)** : [0.일자별 잔액표시] 동일한 일자의 전표는 전표데이터의 마지막 행에 잔액을 표시 / [1.건별(전표별) 잔액표시] 각 전표별로 잔액을 표시
- **④ 우측마우스 > 체크펜** : 색상 옆 체크박스에 체크[V]하면 클릭한 라인에 선택한 색이 반영되며, 취소 옆 체크박스에 체크[V]하면 제거됩니다.

## 6) 계정별원장

> 메뉴위치 : [회계관리] 주요장부 ▶ 계정별원장

### ▣ 개요
현금 계정을 제외한 계정과목의 계정별 원장을 조회 및 출력하는 메뉴로 "계정별, 부서별, 사원별, 현장별, 프로젝트별" 탭으로 구성되어 있습니다.

### ▣ 조회방법

#### 1) 계정별
- **① 당기/전기** : 현재 접속한 기수(당기)와 직전 기수(전기)를 선택하여 데이터를 조회하며, '전기'로 조회한 경우 데이터 수정은 불가능합니다.
- **② 기간** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **③ 계정과목** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회하고자 하는 계정과목을 선택합니다.
  - 전체 계정과목을 조회하고자 하는 경우 별도 입력없이 [Enter]를 클릭합니다.
  - 복수의 계정과목을 조회하고자 하는 경우 물결 '~'을 사용하여 구간을 지정할 수 있습니다. (예: 801~831)
- **④ 구분** : [환경설정]메뉴 공통탭의 계정과목코드체계 > [1. 세목사용(5자리)]로 설정한 경우 [0.과목, 1.세목]을 선택하여 조회합니다. [1.세목]으로 조회 시 사용자가 설정한 계정세목으로 조회가능하며, [0.과목]으로 조회시 합산된 계정과목으로 조회됩니다.
- **⑤ 금액** : 0.전체(별도 조건없이 검색) / 1.차변(전표 입력 시 [3.차변]으로 입력한 데이터 중 특정 금액 설정 조회) / 2.대변(전표 입력 시 [4.대변]으로 입력한 데이터 중 특정 금액 설정 조회)

#### 2) 부서별/사원별/현장별/프로젝트별
- **① 부서/사원/현장/프로젝트** : 조회하고자 하는 관리항목(부서/사원/현장/프로젝트)을 설정하여 조회합니다. 전체 관리항목으로 조회할 경우 코드입력 없이 [Enter] 클릭합니다.

#### 3) 기능키 (툴바)
- **① 행고정/행고정해제** : [행고정]을 클릭시 조회된 첫행부터 마우스 커서가 위치한 행까지 화면이 고정됩니다. 해제 시 [행고정 해제]를 클릭합니다.
- **② 정렬** : [계정성격순] 자산·부채·자본의 계정과목 성격상 증가하는 거래가 우선 정렬, 그 다음 감소되는 거래로 정렬 / [전표번호순] 전표번호순으로 정렬
  - TIP_'전표번호순' 정렬 시 참고사항 : [일반전표입력]의 전표번호는 00001부터 시작, [매입매출전표입력]의 전표번호는 50001부터 시작하므로, [전표번호순]의 경우 [매입매출전표입력]을 먼저 입력했더라도 [일반전표입력] 데이터가 우선 정렬됩니다.

#### 4) 기능키 (더보기(기능모음) 및 마우스 우클릭)
- **① 집계옵션 (F3)** : 데이터 조회시 집계항목인 [월계, 누계]를 화면에 표기할지 여부를 선택합니다.
- **② 건별잔액설정 (Ctrl+Z)** : 거래건별 잔액의 표시 [유/무]와 잔액이 음수일 경우 빨간색 [표시함/안함]를 선택합니다.
- **③ 화면구성 (F4)** : 화면을 구성할 항목을 설정합니다.
- **④ 전표수정 (F7)** : [일반전표입력], [매입매출전표입력]메뉴 실행없이 [계정별원장]메뉴에서 바로 분개를 수정할 수 있습니다. 수정하고자 하는 전표를 더블클릭 하거나, 단축키(F7)을 클릭 시 하단 분개내역이 반영되며, 분개 수정 및 삭제가 가능합니다.
- **⑤ 우측마우스 > 체크펜** : 색상 옆 체크박스에 체크[V]하면 클릭한 라인에 선택한 색이 반영되며, 취소 옆 체크박스에 체크[V]하면 제거됩니다.
- **⑥ 우측마우스 > 계산기 (Ctrl+Q)** : 계산기 기능을 실행합니다.
- **⑦ 우측마우스 > 엑셀변환** : 조회된 화면을 엑셀로 변환할 수 있습니다.

## 7) 매입매출장

> 메뉴위치 : [회계관리] 주요장부 ▶ 매입매출장

### ▣ 개요
부가가치세와 관련된 내역을 기록한 보조 장부로, [매입매출전표입력] 메뉴의 상단부에 입력된 내역을 조회할 수 있습니다. [매입매출장] 메뉴는 [전체/현장별/프로젝트별/부서별/사원별] 탭으로 구성됩니다.

### ▣ 조회방법

#### 1) 전체
- **① 기간** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **② 사업장명** : [회사등록] 메뉴 본지점설정의 [본.지점 여부]가 [본점]이고, [지점 현황]이 등록된 경우, 본점에서는 지점의 [매입매출장]을 조회할 수 있습니다. 본점과 지점 중 조회할 사업장을 선택합니다.
- **③ 조회구분** : [1.개별] 사업장명에서 지정한 특정 사업장의 데이터를 조회 / [2.통합] 사업장명에서 지정한 모든 사업장의 데이터를 통합하여 조회
- **④ 구분** : [0.전체] 구분없이 전체 조회 / [1.매출] 유형이 매출[11.과세, 12.영세, 13.면세, 14.건별 등]인 과세유형 조회 / [2.매입] 유형이 매입[51.과세, 52.영세, 53.면세, 54.불공 등]인 과세유형 조회 / [3.신고서기준] 부가가치세 신고서의 기준에 따라 매입과 매출 내역을 조회(면세 거래나 세금계산서 외 거래 등은 제외)
- **⑤ 조회** : [∨] 버튼 클릭시 하단으로 '조건검색'의 조건 화면이 노출되며, 조건을 설정하여 상세 검색 가능합니다. 조건검색한 경우 분홍색 [∧] 버튼으로 표기됩니다.

#### 2) 현장별/프로젝트별/부서별/사원별
[매입매출전표입력] 메뉴의 상단부 [관리]에서 입력한 관리항목을 기준으로 매입매출전표를 조회합니다.
- **① 기간** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **② 현장/프로젝트/부서/사원** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회할 [현장/프로젝트/부서/사원]을 선택합니다.

#### 3) 기능키
- **① 전자세금** : [전자(세금)계산서 전체] 전자(세금)계산서 발급분을 모두(과세기간 종료일 다음달 11일 이후 전송분 포함) 조회 / [전자(세금)계산서 11일 이내] 과세기간 종료일 다음달 11일 이내 전송 데이터 조회 / [전자(세금)계산서 11일 이후] 과세기간 종료일 다음달 11일 이후 전송 데이터 조회
- **② 종이발행** : [종이발행] 종이 (세금)계산서 조회 / [종이발행 + 전자 11일 이후] 종이 (세금)계산서 발급분과 과세기간 종료일 다음달 11일 이후 전송한 데이터 조회
- **③ 행고정/행고정 해제** : [행고정]을 클릭시 조회된 첫행부터 마우스 커서가 위치한 행까지 화면이 고정됩니다. 고정한 행을 해제 시 [행고정 해제]를 클릭합니다.
- **④ 우측마우스 > 엑셀변환** : 조회된 화면을 엑셀로 변환할 수 있습니다.
- **⑤ 우측마우스 > 계산기(Ctrl+Q)** : 계산기 기능을 실행합니다.

#### 4) 더보기(기능모음)
- **① 화면구성** : 화면에 표시되는 항목을 체크하여 선택합니다.
- **② 집계옵션 (F3)** : 데이터 조회시 집계항목인 [합계, 월계, 분기누계, 반기누계, 누계]를 화면에 표기할지를 선택합니다.
- **③ 주민번호 별표 사용설정** : [사업자(주민)등록번호] 란의 주민번호 뒷자리 '*' 처리 여부를 설정합니다.
- **④ 신고년월 (Ctrl+M) [전체 탭]** : 예정신고 누락분과 내국신용장 사후개설분 데이터를 조회합니다.
- **⑤ 예정누락(Shift+F7) [전체 탭]** : [매입매출전표입력] 메뉴에서 더보기(기능모음)의 [예정누락]을 이용해서 입력된 전표를 조회합니다.
- **⑥ 사후개설 (Ctrl+F12) [전체 탭]** : 내국신용장 사후개설분 전표를 조회합니다.
- **⑦ 전표수정 (F7)** : [매입매출전표입력] 메뉴 실행없이 [매입매출장] 메뉴에서 전표를 바로 수정할 수 있습니다. 전표를 더블클릭 하거나, 단축키(F7)을 클릭 시 하단 분개내역이 반영되어 수정 가능합니다.
- **⑧ 부분별집계 [전체 탭 제외]** : 조회한 기간의 관리(현장, 프로젝트, 부서, 사원)의 부분별 내역을 조회 및 인쇄합니다.
- **⑨ 유형별 전표 집계(F4)** : 조회한 기간의 과세 유형별 집계 내역을 조회 및 인쇄합니다.
- **⑩ 거래처별 집계(Shift+F10)** : 조회한 기간의 거래처별로 유형(1.매출, 2.매입)에 따라 내역을 조회 및 인쇄합니다.

## 8) 총괄원장

> 메뉴위치 : [회계관리] 주요장부 ▶ 총괄원장

### ▣ 개요
거래처 및 관리항목(프로젝트(PJT), 현장, 부서, 사원)에 대해 각 계정과목별 총괄잔액 또는 총괄내용을 조회하는 메뉴입니다.

### ▣ 조회방법

#### 1) 잔액(PJT/현장/거래처/부서/사원)
전표 입력시 설정한 거래처코드, 관리항목(PJT/현장/부서/사원)의 각 계정과목별 잔액을 확인할 수 있습니다.
- **① 일자** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **② 계정과목** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회하고자 하는 계정과목을 선택합니다.
  - 전체 계정과목을 조회하고자 하는 경우 별도 입력없이 [Enter]를 클릭합니다.
  - 복수 계정과목을 조회하고자 하는 경우 물결 '~'을 사용하여 구간을 지정할 수 있습니다. (예: 811~831)
- **③ 구분** : [환경설정]메뉴 공통탭의 계정과목코드체계 > [1. 세목사용(5자리)]로 설정한 경우 [0.과목, 1.세목]을 선택하여 조회합니다. [1.세목]으로 조회 시 사용자가 설정한 계정세목으로 조회가능하며, [0.과목]으로 조회시 합산된 계정과목으로 조회됩니다.
- **④ 프로젝트/현장/거래처/부서/사원** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회하고자 하는 [프로젝트/현장/거래처/부서/사원]을 선택합니다.

#### 2) 내용(PJT/현장/거래처/부서/사원)
전표 입력시 설정한 거래처코드, 관리항목(PJT/현장/부서/사원)의 각 계정과목별 잔액에 대한 상세내역을 확인할 수 있습니다.
- **① 일자** : 조회하고자 하는 기간의 월/일을 입력합니다.
- **② 계정과목** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회하고자 하는 계정과목을 선택합니다.
  - 전체 계정과목을 조회하고자 하는 경우 별도 입력없이 [Enter]를 클릭합니다.
  - 복수 계정과목을 조회하고자 하는 경우 물결 '~'을 사용하여 구간을 지정할 수 있습니다. (예: 811~831)
- **③ 구분** : [환경설정]메뉴 공통탭의 계정과목코드체계 > [1. 세목사용(5자리)]로 설정한 경우 [0.과목, 1.세목]을 선택하여 조회합니다. [1.세목]으로 조회 시 사용자가 설정한 계정세목으로 조회가능하며, [0.과목]으로 조회시 합산된 계정과목으로 조회됩니다.
- **④ 프로젝트/현장/거래처/부서/사원** : [F2] 코드도움 또는 [C] 아이콘을 사용하여 조회하고자 하는 [프로젝트/현장/거래처/부서/사원]을 선택합니다.

## ★ BK 매핑·시사점

1. **장부 = 전표의 파생 뷰** : 모든 장부는 전표 입력 데이터를 집계/필터한 결과이며 별도 입력이 없음. BK는 전표(`Journal`/`JournalLine`)를 단일 원천으로 두고 장부를 조회 쿼리/머티리얼라이즈드 뷰로 구현.

2. **전표번호 체계** : [일반전표입력]=1~50000, [매입매출전표입력]=50001~99999. 정렬 시 일반전표가 우선 정렬되는 규칙은 BK 전표 채번/정렬 정책에 반영.

3. **장부에서 직접 전표수정(F7)** : 분개장·현금출납장·계정별원장·일월계표(원장조회)·매입매출장 모두 메뉴 실행 없이 즉시 분개 수정·삭제 가능. BK는 장부 행→원전표 드릴다운 + 인라인 편집(권한·기간잠금 검증 포함)으로 구현. 단 분개장은 현금합산 적용안함 상태에서만 수정 가능.

4. **세목/과목 토글** : 환경설정 계정과목코드체계 [1.세목사용(5자리)] 설정 시 [0.과목/1.세목] 조회 분기. BK 세목(5자리) vs 과목(3자리) 모델과 정합 — 장부 조회 파라미터로 구현.

5. **다사업장·다차원 필터** : 매입매출장은 본/지점 사업장 개별·통합 조회, 모든 원장이 현장/프로젝트/부서/사원 탭 제공. BK 6 관리차원 모델로 충족. 매입매출장 구분 [3.신고서기준]은 부가세 신고 대사 모드.

6. **현금출납장 입출 반영 규칙** : [2.입금/3.차변(현금)]→입금, [1.출금/4.대변(현금)]→출금. JV-02/03 현금 간편입력과 직접 연동. AUX 현금출납장 정의와 일치.

7. **부가세 연계** : 매입매출장은 전자세금/종이발행/예정누락/사후개설/신고년월 등 부가세 신고 요소를 직접 다룸 → 08~11 부가세 서식과 연결. 유형별·거래처별 집계 기능은 BK 부가세 집계 후보.

8. **공통 3요소** : 모든 장부 조회는 ① 기간·사업장·관리차원 필터, ② 집계옵션(일계/월계/분기누계/반기누계/누계), ③ 엑셀변환·체크펜·계산기·인쇄를 공통 제공 → BK 장부 컴포넌트의 표준 기능셋으로 채택.
